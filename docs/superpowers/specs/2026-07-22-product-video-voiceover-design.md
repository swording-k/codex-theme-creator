# Codex Theme Creator 产品视频旁白设计

## 目标

为现有 36.2 秒横屏产品视频制作一条可独立使用的中文男声旁白音轨。音轨不含 BGM，方便发布到抖音时另配平台音乐。

## 声音方向

- 年轻、沉稳、自然的科技产品男声。
- 克制表达，不使用夸张广告腔。
- 旁白响度统一，峰值保留安全余量。
- 句间停顿跟随现有镜头呼吸位，结尾品牌句说完后留白。

## 定稿文案

1. 一张喜欢的画面，不该只是一张壁纸。
2. Codex Theme Creator，让它成为一套完整的工作台主题。
3. 选择主题，即时切换。
4. 模糊、压暗、透明度，每个细节都能调整。
5. 这个功能通过产品内置的 Skill，一键复制提示词，再交给 Codex。
6. 无论是一句话，还是一张参考图，都能生成属于你的主题。
7. 所有主题保存在本机，也能随时导入和导出。
8. 想回到原版，一键恢复。
9. Codex Theme Creator，把喜欢的画面，变成你的 Codex 工作台。

## 交付

- WAV 主文件：48 kHz、单声道 PCM，便于继续剪辑。
- MP3 便携版：便于直接试听和导入剪映或抖音。
- 不混入 BGM，不修改原视频。

## 验收

- 无爆音、削波和明显底噪。
- 综合响度适合人声素材，真峰值不高于 -1 dBFS。
- 文案准确包含 App 内置 Skill、一键复制提示词、交给 Codex 的产品链路。
- 总时长不超过现有 36.2 秒视频，并保留自然停顿。
