# Porsche GT3 RS

适配 `Fei-Away/Codex-Dream-Skin` 的 GT 赛车控制台主题。

## 文件

- `theme.json`: 可直接安装的 schema v2 主题元数据。
- `source-theme.json`: Codex Theme Creator 的源主题配置。
- `background.jpg`: 2560 x 1440 背景图。
- `preview.jpg`: 真实 Codex Desktop 截图预览。

## 设计

- 深色赛道维修区与雨后地面反射，右侧保留主车视觉焦点。
- 左侧使用低细节暗部，保证 Codex 侧栏、项目列表和聊天入口可读。
- UI 使用 `gt-control` 风格，包含橙色选中态、烟熏玻璃输入框和赛事控制台感。

## 使用

```bash
./scripts/install-local-theme.sh preset-porsche-gt3rs
~/.codex/codex-dream-skin-studio/scripts/switch-theme-macos.sh --id preset-porsche-gt3rs
```
