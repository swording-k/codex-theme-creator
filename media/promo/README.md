# 宣传素材

这个目录放可以公开发布的主题宣传图，适合用于 GitHub README、自媒体文章、视频封面或演示脚本。

每个主题目录里通常有两类图片：

- `preview.png`：主题预览图，优先用于文章配图和视频截图。
- `background.jpg`：主题背景图，适合展示壁纸质感，也可作为封面素材二次排版。

当前素材：

- `porsche-gt3rs`：GT 赛车 / 赛道控制台氛围。
- `iron-discipline`：高级铁馆 / 健身风格。
- `rainforest-focus`：雨中森林 / 编程专注氛围。
- `alpine-lake-desk`：湖泊雪山 / 桌面工作氛围。

注意：`private-themes/` 是私人本地主题目录，不会进入仓库；里面可能包含本地 Codex 项目名或聊天内容，不适合公开发布。
